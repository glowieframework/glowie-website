# Installation
To install Glowie and start a fresh new project, use:

```plaintext
composer create-project glowieframework/glowie myproject
```

This will create a new Glowie application inside `myproject` folder.

That's all. Incredible and simple, isn't it?

### Updating from an earlier version
Updating Glowie is as simple as installing it. Just navigate to your application directory and run:

```plaintext
composer update
```

**Important!** Glowie follows [Semantic Versioning](https://semver.org) standards, so don't forget to read the changelog and be aware of all breaking changes before running any major updates.
